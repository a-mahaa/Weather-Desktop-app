# Weather-Desktop-app

Description:
This project is a weather application built with Python and a GUI interface. Users can enter a city name to view current weather conditions, temperature, humidity, wind speed, sunrise, and sunset information.

Language Used:
Python

Libraries Used:
Tkinter
Requests
Time

Features:
- Search weather by city
- Current temperature display
- Minimum and maximum temperature
- Humidity and pressure data
- Wind speed information
- Sunrise and sunset times
- GUI desktop application
